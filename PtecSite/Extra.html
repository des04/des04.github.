Spans

Spans are things people use to group inline elements in a document. For example, this text is different than the preceding text because it is within a span div that has stylized the text to be purple. Selectors for Anchors

An anchor put very simply is a link. Now, a regular default link (as you can see) looks quite bland. So, using HTML and CSS at PTEC we were taught how to stylize links to make them look much more asthetically pleasing. An example of a stylized set of links would be the links at the top of this page in the nav bar. These links use the hover attribute, but sadly, none of these links use the focus for visited attributes. The focus attribute allows for links to change colour while being clicked. Here is an example of one that does this (you can tell because it changes to aqua while being clicked). The visited attribute does the same thing as the focus attribute, only after the click happens. For example, have you ever heard of a "visited link"? In google, whenever you've already clicked a link, it'll be a different colour. This is because it uses the visited attribute.
